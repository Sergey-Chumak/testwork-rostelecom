Задание

Использовать https://jsonplaceholder.typicode.com/ или похожие сервисы как тестовый REST API

Язык JavaScript (REACT JS)

 

1) Сделать WEB запросы к сайту REST API

2) На основании полученных данных отрисовать на HTML странице информацию с применением стилей (CSS, LESS)

3) Сделать поиск по полученным данным

4) Сделать сортировку по данным

5) Прочитать файл json.txt, распарсить, получить корректный объект с html разметкой. Вывести в формы всю информацию из JSON (DIV, Section и тд – формы просмотра) на странице без потерь стилей и контента
